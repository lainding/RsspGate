RsspGate
---------------------------------------------
**功能：**
进行协议转换与转发

+--------+   +--------+   +--------+   +--------+   +--------+ 
|        |   |        |   |        |   |        |   |        |
|        |   |        |   |        |   |        |   |        |
| Device |-->|  Gate  |-->|  App   |-->|  Gate  |-->| Device |
|        |   |        |   |        |   |        |   |        |
|        |   |        |   |        |   |        |   |        |
+--------+   +--------+   +--------+   +--------+   +--------+

**配置**
配置`config.json`包括3部分：分别定义设备，端口和路由方式
1. 设备`devices`
一个远端端口被定义为一个设备。
`name`： 设备名称（**必选项**）
